# Rust_furnace_helper
It will help you with filling up your furnaces in real time.


requirements: cv2, 
              mss, 
              PIL, 
              pytesseract, 
              keyboard.
              
How to use:
          1) start program
          2) open inventory in rust
          3) chose ore and amount of it.
