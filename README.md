# Rust_furnace_helper
It will help you with filling up your furnaces in real time.


requirements: cv2, 
              mss, 
              PIL, 
              pytesseract, 
              keyboard.
              
How to use:

          1) Start program
          
          2) Open inventory in rust
          
          3) Chose ore and amount of it. 123 metal ore in example.
